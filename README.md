# Dreamleaf Learning (Team B Final Project)
## Project Summary
Dreamleaf Learning is a English learning app that allows the user to play games related
to English comprehension. The games include:
- **Fill in the Sentence**: Select the word that best fits in the incomplete sentence.
- **Listen and Choose**: Listen to a spoken word, and choose the word that was spoken.
- **Mixed Sentence**: With a list of words, rearrange the words to create a sentence that makes sense.
- **Picture Match**: Pick the word matching the shown picture.
- **Rhyming Words**: Given a word, pick a word from a list of words that rhymes.
